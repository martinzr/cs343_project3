  • Our site is deployed at: https://martinzr.github.io/cs343_project3/
  • Our site uses 2 different API's: WatchMode and uNoGS.
  • The purpose of our site is to allow the user to search for a movie or tv show, which will provide buttons matching the search results.
    By clicking a search result, the site then lists which countries Netflix offers that movie/show.
  • Our site uses a search function to look up a movie/tv show.
  • The results of the search function are displayed as buttons for each result.
  • The user can interact with the results of the search by clicking on any result and receiving a list of countries where Netflix offers the movie/tv show.
